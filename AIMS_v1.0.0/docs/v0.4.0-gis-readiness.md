# AIMS v0.4.0 GIS Readiness

## Purpose

v0.4.0 adds a GIS preflight layer on top of the ADF model.

The goal is not to replace a professional GIS topology engine. The goal is to detect obvious CAD-to-GIS risks before export.

## Checks

- Missing geometry
- Unknown categories
- Missing level/floor information for spatial entities
- Open boundaries
- Zero-area polygons
- Duplicate geometry
- Bounding-box overlap warnings

## Outputs

- SQLite database
- GeoJSON FeatureCollection
- Markdown QA/QC report with BIM and GIS readiness scores
