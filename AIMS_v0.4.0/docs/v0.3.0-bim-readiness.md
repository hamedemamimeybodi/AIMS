# AIMS v0.3.0 BIM Readiness Architecture

v0.3.0 adds a BIM readiness layer over the existing ADF pipeline.

## Pipeline

```text
DXF
 ↓
DXF Parser
 ↓
ADF Normalizer
 ↓
Architectural Validation
 ↓
BIM/IFC Mapping
 ↓
BIM Readiness Validation
 ↓
SQLite + Markdown Report
```

## Design Rule

No downstream module should depend directly on DXF. BIM, GIS, reports, and databases consume ADF only.

## BIM Mapping Priority

1. Block name mapping
2. Layer mapping
3. Category mapping
4. Default category fallback

## Current Limitations

- No IFC file export yet.
- No Revit family generation yet.
- No wall thickness inference yet.
- No room boundary graph solver yet.

Those come later. First the data needs a skeleton, because throwing IFC at bad CAD is just exporting confusion in a more expensive format.
