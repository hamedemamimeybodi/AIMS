# AIMS v0.7.0 - Reporting & Dashboard Foundation

## هدف

نسخه v0.7.0 لایه گزارش‌گیری پروژه را از یک خروجی Markdown ساده به یک زیرسیستم گزارش‌گیری چندفرمتی تبدیل می‌کند.

## خروجی‌های جدید

- `reports/aims_report.md` برای خواندن سریع در GitHub
- `reports/aims_report.html` برای مرور بصری و چاپ بهتر
- `reports/aims_report.json` برای مصرف ماشینی، داشبورد، CI/CD و ابزارهای بعدی
- `reports/aims_output.geojson` برای GIS
- `reports/aims_output.sqlite` برای ذخیره ساختاریافته

## ماژول‌های اضافه‌شده

```text
packages/aims_report/
├── summary.py
├── json_report.py
├── html_report.py
└── markdown_report.py
```

## Scorecard

گزارش جدید یک Scorecard مرکزی دارد:

```text
Architectural Quality
BIM Readiness
GIS Readiness
Rule Engine
Plugin System
Overall Score
```

`Overall Score` میانگین کنترل‌شده همین امتیازهاست و فعلاً وزن‌دهی پیشرفته ندارد. وزن‌دهی حرفه‌ای در نسخه‌های بعدی اضافه می‌شود، چون ظاهراً حتی امتیاز دادن هم باید بروکراسی داشته باشد.

## CLI

نمونه اجرا:

```bash
aims audit examples/simple_site.dxf \
  --out reports/aims_report.md \
  --html reports/aims_report.html \
  --json-report reports/aims_report.json
```

- اجرای تست خودکار

- اجرای audit روی نمونه‌ها
