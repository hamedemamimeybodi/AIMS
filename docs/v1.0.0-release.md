# AIMS v1.0.0 Professional Release Candidate

AIMS v1.0.0 consolidates the previous foundation releases into a single manual-push package suitable for repository use, CI validation, and repeatable CAD/BIM/GIS QA/QC demonstrations.

## Scope

- DXF to ADF normalization
- architectural QA/QC validation
- BIM readiness scoring
- GIS readiness scoring
- YAML rule engine
- plugin execution
- OpenBIM validation
- deterministic IFC-like export
- SQLite, GeoJSON, Markdown, HTML, and JSON outputs
- project health validation
- release gate validation
- GitHub Actions CI workflow

## Non-goals

This release does not claim to be a full Revit replacement, a complete IFC authoring engine, or a magical cure for chaotic CAD drafting. Full geometry-rich IFC export should later use IfcOpenShell.

## Release commands

```bash
python -m aims.cli validate --version 1.0.0
python -m aims.cli release-check --version 1.0.0
pytest
```
