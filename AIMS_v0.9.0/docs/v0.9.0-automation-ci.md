# AIMS v0.9.0 - Automation & CI Foundation

## هدف

این نسخه برای آماده‌سازی پروژه جهت توسعه پایدار و رسیدن به v1.0 ساخته شده است. تمرکز آن روی automation، smoke test، CI و release manifest است.

## قابلیت‌ها

- GitHub Actions CI workflow
- اجرای pytest روی Python 3.11 و 3.12
- اجرای smoke audit روی `examples/simple_site.dxf`
- دستور `aims validate`
- تولید `reports/release_manifest.json`
- اسکریپت‌های محلی smoke برای Bash و PowerShell

## دستورات

```bash
python -m aims.cli validate
python -m aims.cli audit examples/simple_site.dxf
```

## مسیرهای مهم

```text
.github/workflows/ci.yml
ci/smoke.sh
ci/smoke.ps1
packages/aims_automation/
reports/release_manifest.json
```

## جایگاه در Roadmap

این نسخه آخرین foundation قبل از v1.0 است. بعد از آن تمرکز باید روی hardening، مستندسازی، sample data بیشتر، و بسته‌بندی release باشد.
